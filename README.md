# measureLength

実験で使うカメラの実世界座標変換の値がどれだけずれているか測るプログラム.

# MODE
- w: 前回クリックした座標から現在クリックした座標までの実世界x軸距離を出力
- h : 前回クリックした座標から現在クリックした座標までの実世界y軸距離を出力
- x : 現在クリックした座標の実世界x座標を取得
- y : 現在クリックした座標の実世界y座標を取得
- q : プログラムを終了する
